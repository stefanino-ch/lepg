LEP GUI (Tcl/Tk) in development
-------------------------------

- Linux: Install tclsh interpreter in your Linux computer
- MAC OSX: Run tcl insterpreter form terminal, or use wish
- Windows: Install "wish" interpreter

Remove ".txt" in ".tcl.txt" file extensions
Put all .tcl and .txt files provided in the same directory
Make dir "lep" and put inside leparagliding.txt

Run main script:  

leg.tcl

Example, in Linux:

$ chmod +x leg.tcl
$ ./leg.tcl
