Prinzipielle Funktionen in den Eingabefenstern

Eine Einführung zu den prinzipiellen Funktionen der Eingabefenster.

Ein Eingabefenster wird prinzipiell über 3 Schaltflächen gesteuert:

  • [Apply]
  • [OK]
  • [Cancel]

Wird [Apply] angeklickt passieren 2 Dinge:

  • Alle Elemente des Fensters werden aktualisiert.
  • Die aktuellen Daten werden in das Hauptfenster zurück synchronisiert. Das bedeutet dass auch das Hauptfenster und weitere abhängige Eingabefenster ebenfalls aktualisiert werden.

Auch [OK] löst zwei Aktionen aus:

  • Die aktuellen Daten werden in das Hauptfenster zurück synchronisiert. Das bedeutet dass auch das Hauptfenster und weitere abhängige Eingabefenster ebenfalls aktualisiert werden.
  • Danach wird das Eingabefenster geschlossen.

[Cancel] schliesst das Eingabefenster. Alle Datenänderungen seit dem letzten clicken von [Apply] sind verloren!
