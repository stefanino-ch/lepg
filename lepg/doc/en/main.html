English manual

Know the general principles is need...

3 Main components are used to created a complete glider plan:

  • GUI: allows all data management in a user friendly way
  • Geometry-Processor: does in the background the main calculations for the outile geometry of the wing
  • Wing-Processor: does then at the final end all the detailed calculations and creates all plans

The manual does just explain the GUI and how to start both processors.

More explanations you can find on the developer website: Laboratory d'envol => Software => pre-processor

Steps from the idea to the final glider plan:

  1. Define the wing outline in the Geometry window
  2. Run the Geometry-Processor
  3. Continue with the detailed definition of the wing
  4. Run the Wing-Processor
  5. Start to build your wing
