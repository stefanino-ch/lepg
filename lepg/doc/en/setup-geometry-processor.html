Setup Geometry Processor window

Detailed explanation about the Setup Geometry Processor window.

The Geometry Processor is an external executable. It is started from lepg. As the location or the version of the Gemetry Processor can change you must configure initially the right executable to use.

Usually the lepg package conatains an compatible Geometry Processor. It is saved in a subdirectory called "pre". The name of the executable will be equal or similar to "pre-1.4.exe".
